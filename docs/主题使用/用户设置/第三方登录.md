# 第三方登录
作者：[阿城](https://www.hidesg.ink)

:::danger
说明：第三方登录使用 OAuth，通过固定回调地址完成授权。请在对应平台后台把“回调地址”设置为下面的 URL（严格一致）。
:::

## 启用第三方登录
![](https://a1.boltp.com/2026/03/25/69c358c1b83c3.png)
开启后，支持用户使用第三方登录。

## 允许自动注册新用户
![](https://a1.boltp.com/2026/03/25/69c358f6bb7b1.png)
开启后，未绑定的第三方账号会自动创建站点用户并登录。

## GitHub 登录
![](https://a1.boltp.com/2026/03/25/69c35bd5663a6.png)

**启用 GitHub**
总开关，开启后 GitHub 登录功能生效。

**Client ID**
GitHub 应用的客户端 ID，需在 GitHub 开发者平台创建应用后获取。

**Client Secret**
GitHub 应用的客户端密钥，同样在 GitHub 开发者平台创建后获取，需严格保密，不可泄露。

## QQ 登录（QQ 互联）
![](https://a1.boltp.com/2026/03/25/69c35b624c21a.png)

**启用 QQ 登录**
总开关，开启后 QQ 登录功能生效。

**App ID**
QQ 应用的客户端 ID，需在 QQ 互联平台创建应用后获取。

**App Key**
QQ 应用的客户端密钥，同样在 QQ 互联平台创建后获取，需严格保密，不可泄露。
