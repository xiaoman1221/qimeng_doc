# 登录 / 注册
作者：[阿城](https://blog.morehouse-s.com/)

## 基础设置
![](https://a1.boltp.com/2026/02/27/69a10d4dc3648.png)

### 允许注册
开启时：用户可以正常访问注册页面，完成账号注册流程。
关闭时：前端注册入口会被隐藏，彻底关闭新用户注册通道。

### 默认勾选 “记住账号”
在登录页面，“记住账号” 选项会被默认勾选。

## 跳转设置

### 登录成功跳转地址



用户完成登录后，系统会自动跳转到个人中心页面（/user）。
![](https://a1.boltp.com/2026/02/27/69a10fbbc34f3.png)

### 注册成功跳转地址
新用户完成注册后，系统会自动跳转到个人中心页面（/user）。

## 安全增强
![](https://a1.boltp.com/2026/02/27/69a1110f304f4.png)

### 启用邮箱验证码
开启后，登录和注册时必须输入邮箱验证码才能完成操作；关闭后，不显示验证码输入框。

### 启用邮箱验证（链接验证）
开启后，注册成功会发送验证邮件；未验证邮箱将被禁止登录。

### 启用频率限制（防爆破）
对登录 / 注册接口做简单频率限制（按 IP）。

### 窗口内最大尝试次数
在指定的时间窗口内，单个 IP 最多允许尝试 10 次登录 / 注册操作。

### 窗口时长（秒）
频率限制的统计时间窗口为 5 分钟。
