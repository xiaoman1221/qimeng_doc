# 签到设置
作者：[阿城](https://www.hidesg.ink/)

## 启用签到功能
![](https://a1.boltp.com/2026/03/13/69b409ec31058.png)
开启后，用户端将显示签到入口；关闭后，入口隐藏。

## 奖励发放到
![](https://a1.boltp.com/2026/03/13/69b39a6f2506d.png)
这是平台奖励发放渠道的配置项，用于指定签到、任务等用户行为所获奖励的入账类型。

## 签到规则链接
![](https://a1.boltp.com/2026/03/13/69b39a991b600.png)
用于在签到入口旁提供详细规则说明。在用户端的签到按钮或弹窗旁，显示 “查看规则” 入口，点击后跳转到指定页面，展示完整的签到规则、奖励机制、补签说明等内容。

## 7天奖励配置
![](https://a1.boltp.com/2026/03/13/69b39adace7e8.png)
7 天连续签到的额外奖励配置项，用于激励用户完成一周的连续打卡。

## 签到弹窗样式

### 弹窗破界图
![](https://a1.boltp.com/2026/03/13/69b39af796e9d.png)
签到弹窗右上角的 “破界图” 配置项，用于增强签到弹窗的视觉吸引力。
