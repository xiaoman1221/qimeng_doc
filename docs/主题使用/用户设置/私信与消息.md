# 私信与消息
作者：[阿城](https://www.hidesg.ink)

## 消息通知缓存天数
![](https://a1.boltp.com/2026/03/25/69c35784175ab.png)
超过该天数的私信记录与站内消息通知会被自动删除；设置为 0 表示不自动清理。

## 消息图片大小上限
![](https://a1.boltp.com/2026/03/25/69c3579e02f38.png)
单张图片最大体积，超出后前后端都会拒绝上传。

## 允许的图片类型
![](https://a1.boltp.com/2026/03/25/69c357b3526ac.png)
使用英文逗号分隔，例如：jpg,jpeg,png,gif,webp,avif
