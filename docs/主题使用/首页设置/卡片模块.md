## 普通卡片列表
![](https://a1.boltp.com/2026/03/25/69c3490a03d05.png)
这是用于设置页面中内容卡片视觉外观的配置选项，通过切换样式来满足不同的设计需求。

## 桌面端一行显示
![](https://a1.boltp.com/2026/03/25/69c34943b0244.png)
这是一个内容布局 / 排版的自适应设置项，作用是控制不同设备上，一行能展示多少个内容卡片，让页面在桌面端、移动端都有更舒适的浏览体验。


## 异形卡片列表
![](https://a1.boltp.com/2026/03/25/69c34773d9f6c.png)
这是异形卡片样式下的具体内容配置面板，用于创建带视觉特效的可跳转卡片。

## 桌面端显示数量
![](https://a1.boltp.com/2026/03/25/69c347e0e0902.png)
这是用于控制桌面端页面中卡片 / 内容项一行展示个数的配置选项，用来优化页面布局和视觉体验。

## 移动端显示数量
![](https://a1.boltp.com/2026/03/25/69c3487328429.png)
这是用于控制移动端页面中卡片 / 内容项一行展示个数的配置选项，同时会对超出数量的内容进行隐藏，以适配手机等小屏设备的浏览体验。
