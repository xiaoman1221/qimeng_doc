# 幻灯片设置
作者：[阿城](https://blog.morehouse-s.com/)

## 启用幻灯片模块
![5.41.1.png](../../../static/img/docs/5/5.41.1.png)
启用后将在前台显示幻灯片。

幻灯片一共有三种布局类型：**普通布局 三合一布局 上大图下卡片** 接下来将各个布局配置样式总结如下。

### 全屏布局
![5.41.2.png](../../../static/img/docs/5/5.41.2.png)
桌面端高度指的是幻灯片在电脑网页端展示的高度，默认高度为600px。可以根据自己喜好自由调整。


![5.41.3.png](../../../static/img/docs/5/5.41.3.png)
移动端高度指的是幻灯片在手机网页端展示的高度，默认高度为200px。可以根据自己喜好自由调整。


![5.41.4.png](../../../static/img/docs/5/5.41.4.png)
素材类型：可以选择 图片 / 视频
图片：选择你喜欢的图片上传即可。
链接地址：输入你想跳转的链接。没有请留空。

**全屏效果**
![5.41.5.png](../../../static/img/docs/5/5.41.5.png)

### 三合一布局
![5.41.6.png](../../../static/img/docs/5/5.41.6.png)
左侧大图：选择你喜欢的图片上传即可。
左侧链接：输入你想跳转的链接。没有请留空。

右侧上图：选择你喜欢的图片上传即可。
右侧上链接：输入你想跳转的链接。没有请留空。

右侧下图：选择你喜欢的图片上传即可。
右侧下链接：输入你想跳转的链接。没有请留空。

**三合一效果**
![5.41.7.png](../../../static/img/docs/5/5.41.7.png)

### 上大图下卡片
![5.41.8.png](../../../static/img/docs/5/5.41.8.png)
大图（图片）：选择你喜欢的图片上传即可。
链接地址：输入你想跳转的链接。没有请留空。

![5.41.9.png](../../../static/img/docs/5/5.41.9.png)
卡片图片：选择你喜欢的图片上传即可。
链接地址：输入你想跳转的链接。没有请留空。


![5.41.10.png](../../../static/img/docs/5/5.41.10.png)
桌面端一行显示默认4个，可以根据自己喜好自由调整。

**普通卡片效果**
![5.41.11.png](../../../static/img/docs/5/5.41.11.png)
