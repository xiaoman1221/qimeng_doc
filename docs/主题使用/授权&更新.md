# 授权&更新
## 状态查看
可以查看主题授权情况。
![1](https://a1.boltp.com/2026/02/23/699c670a64810.png)
## 主题授权
可验证主题的授权码，输入授权码之后保存配置。再点击授权，即可授权成功。
![1](https://a1.boltp.com/2026/02/23/699c670a65a19.png)
## 在线更新
可以查看主题版本，检查新版本。更新版本需先勾选**我已完成备份，确认继续更新**在点击**立即更新**::: danger紧急恢复请在重大问题需要回档再使用，无故使用导致配置缺失的我们不负任何责任。:::

![1](https://a1.boltp.com/2026/02/23/699c4f4d8d86f.png)
