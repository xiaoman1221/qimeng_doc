# 工单设置
作者：[阿城](https://www.hidesg.ink)

## 启用用户工单中心
![](https://a1.boltp.com/2026/03/25/69c366bf395a3.png)
在用户中心中显示工单页面，允许用户提交与查看自己的工单。

## 工单页标题
![](https://a1.boltp.com/2026/03/25/69c366e691160.png)
这是社区后台的工单 / 客服页面标题配置项，用于自定义用户提交工单页面的标题。

## 工单页说明文案
![](https://a1.boltp.com/2026/03/25/69c367576729d.png)
这是社区后台的工单 / 客服页面说明文案配置项，用于在用户提交工单的页面展示引导说明。

## 用户中心最多显示工单数
![](https://a1.boltp.com/2026/03/25/69c367889217e.png)
这是社区后台工单系统的前端展示数量配置项，用于限制用户个人中心页面最多展示的历史工单条数，核心是优化页面加载速度与用户体验。

## 允许上传截图
![](https://a1.boltp.com/2026/03/25/69c367d0a9c07.png)
这是工单系统的附件上传权限开关，用于控制用户提交工单时是否可以上传截图 / 图片附件。

## 管理员回复时发送邮箱通知
![](https://a1.boltp.com/2026/03/25/69c3682037abb.png)
这是工单系统的邮件通知开关，用于控制管理员回复工单时，是否自动给提交工单的用户发送邮件提醒。

## 管理员回复/状态变更时发送站内通知
![](https://a1.boltp.com/2026/03/25/69c368971ca1e.png)
这是工单系统的站内消息通知开关，用于控制工单状态更新时，是否给用户推送站内提醒。
