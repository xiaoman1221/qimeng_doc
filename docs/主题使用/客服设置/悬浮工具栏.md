# 悬浮工具栏
作者：[阿城](https://www.hidesg.ink/)

## 启用右侧悬浮工具栏
![](https://a1.boltp.com/2026/03/25/69c36d1c008f7.png)
这是在线客服系统的界面展示开关，用于控制是否在页面右侧展示固定悬浮的客服入口工具栏。

## 客服面板标题
![](https://a1.boltp.com/2026/03/25/69c36e03efc6c.png)
这是在线客服系统的聊天窗口标题自定义项，用于设置用户打开客服面板时，窗口顶部显示的标题文案。

## 客服面板副标题
![](https://a1.boltp.com/2026/03/25/69c36e48d1539.png)
这是在线客服系统的聊天窗口副标题自定义项，用于在主标题下方补充服务说明。

## 在线时间
![](https://a1.boltp.com/2026/03/25/69c36e81e2064.png)
这是在线客服系统的服务时段自定义项，用于在客服面板展示工作室的人工在线时间。

## 客服说明
![](https://a1.boltp.com/2026/03/25/69c36ebba4ac2.png)
这是在线客服系统的服务介绍自定义模块，用于在客服面板展示详细的服务说明。

## 微信号
![](https://a1.boltp.com/2026/03/25/69c36f0486a97.png)
这是在线客服系统的私域联系方式配置项，用于在客服面板展示工作室的微信号，方便用户添加微信进行深度沟通、合作对接。

## QQ
![](https://a1.boltp.com/2026/03/25/69c36f81be300.png)
这是在线客服系统的多渠道联系方式配置项，用于在客服面板展示工作室的 QQ 号，补充微信 / 在线客服的服务渠道。

## 联系电话
![](https://a1.boltp.com/2026/03/25/69c3703428674.png)
这是在线客服系统的电话联系方式配置项，用于在客服面板展示工作室的联系电话，满足用户电话咨询、紧急沟通的需求。

## 客服邮箱
![](https://a1.boltp.com/2026/03/25/69c3706891ce2.png)
这是在线客服系统的邮件服务配置项，用于在客服面板展示工作室的官方邮箱，满足用户正式沟通、文件传输、长期跟进的需求。

## 工具栏项目
![](https://a1.boltp.com/2026/03/25/69c3709fb312a.png)

**类型**
用于关联在线客服聊天系统，用户点击入口直接唤起客服面板。

**标题**
自定义工具栏入口的显示文案，用户在悬浮按钮 / 面板中看到的名称，建议简洁明确、贴合工作室定位。

**图标**
可自定义入口图标，默认是客服耳机图标，也可添加工作室专属图标，强化品牌辨识度
