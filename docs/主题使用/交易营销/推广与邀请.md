# 推广与营销
作者：[阿城](https://www.hidesg.ink/)

## 启用推广与邀请系统
![](https://a1.boltp.com/2026/03/25/69c37265e01e1.png)
这是在线客服系统的营销拓展功能开关，用于激活客服端的推广、邀请、引流相关功能。

## 下级消费返利
![](https://a1.boltp.com/2026/03/25/69c372c49d15e.png)
### 启用下级消费返利
开启后，用户通过邀请链接注册 / 消费，邀请人（上级）可获得对应返利。
### 邀请归因有效天数
设置用户点击邀请链接后，上下级归属关系的保留时长，到期后关系自动失效。
### 参与返利的消费类型
勾选哪些付费场景可触发返利，精准控制返利范围。
### 上级返利规则
点击「添加规则」，自定义返利比例（如消费金额的 5%-20%）、阶梯返利等。

## 邀请码
![](https://a1.boltp.com/2026/03/25/69c373325d1ec.png)
### 启用邀请码绑定
激活邀请码功能。

### 注册时必须填写邀请码
开启后，用户注册账号时强制要求填写邀请码，否则无法完成注册；关闭则为选填。

## 返利提现
![](https://a1.boltp.com/2026/03/25/69c3737b543c7.png)
### 启用返利提现
开启后，邀请人（上级）可将获得的返利提现到个人账户；关闭则无法提现，返利仅可用于账户消费。
### 最低提现金额
设置用户提现的最低门槛，避免频繁提现。
### 可用提现方式
勾选支持的提现渠道，用户可选择对应方式提现。
### 提现说明
自定义提现规则说明，用户在提现页可见，明确提现时效、到账时间、规则等。
