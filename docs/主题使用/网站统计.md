# 网站统计
作者：[阿城](https://blog.morehouse-s.com/)

## 启用统计模块
![](https://a1.boltp.com/2026/03/13/69b376801910c.png)
开启后，用户端将显示；关闭后，入口隐藏。

## 显示的统计项
![](https://a1.boltp.com/2026/03/13/69b3771dc8bd5.png)
可随时调整在前台的网站统计数据。

## 统计项显示顺序
![](https://a1.boltp.com/2026/03/13/69b377706bbe0.png)
该界面用于管理后台数据看板的指标展示清单与排序，采用 “启用 / 禁用” 双列布局，支持通过拖拽调整启用项的展示优先级。

## 建站日期
![](https://a1.boltp.com/2026/03/13/69b378085ae96.png)
用于计算网站运营天数。

## 缓存时长（小时） 
![](https://a1.boltp.com/2026/03/13/69b3782c2562b.png)
该配置项用于设置统计数据的缓存有效期，平衡数据实时性与服务器性能负载。
