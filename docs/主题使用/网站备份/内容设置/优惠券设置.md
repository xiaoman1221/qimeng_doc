# 优惠券设置
作者：[阿城](https://blog.morehouse-s.com/)

## 启用优惠券功能
![](https://a1.boltp.com/2026/02/26/69a02f5d07333.png)
这是整个模块的总开关。 开启后，用户就可以在平台上领取和使用优惠券，用于抵扣消费金额或享受特定优惠。  关闭后，整个模块会被隐藏，前端页面看不到任何相关内容。
## 优惠券列表
![](https://a1.boltp.com/2026/02/26/69a032760df63.png)
### UID
这是这张优惠券在系统内的唯一标识符，用于后台管理和数据追踪，用户在前端不会看到。
### 启用
这是这张优惠券的独立开关：
开启后，这张优惠券才会生效，用户可以领取和使用；
关闭后，这张优惠券会被隐藏或失效，用户无法操作。
### 优惠券类型
用于选择优惠的计算方式
满减券（固定金额）：满足条件后，直接减免固定金额；
其他常见类型还包括折扣券（百分比）、VIP专享等。
### 满减金额
设置这张满减券可以减免的具体金额，单位为元。
### 最低消费金额
设置使用这张优惠券需要满足的最低订单金额。如设置为 0 元，表示这是一张无门槛优惠券，用户无需消费即可使用。
### 优惠券标题
设置这张优惠券在前端显示的名称（如 “10 元无门槛券”），方便用户快速识别。
### 优惠券描述
用于填写优惠券的详细说明，例如使用范围、有效期、特殊限制等，帮助用户了解规则。
### 开始时间
设置这张优惠券开始生效的时间。在这个时间之前，用户无法领取或使用这张券。

![](https://a1.boltp.com/2026/02/26/69a03395b540d.png)
### 过期时间
设置优惠券失效的具体时间。
到达这个时间后，优惠券将自动失效，用户无法再领取或使用。
这是控制优惠活动周期的关键参数。
### 库存数量
限制优惠券可被领取的总张数。
填写数字后，当领取数量达到这个上限，优惠券就会被抢完，无法再领取。
留空则表示不限制领取数量，所有符合条件的用户都可以领取。
### 仅 VIP 可用
这是一个用户群体筛选开关。
开启后，只有 VIP 用户才能领取和使用这张优惠券，用于对核心用户进行专属回馈。
关闭后，所有用户均可领取。
### 限时标签
在前端页面增加视觉提示。
开启后，这张优惠券的展示卡片上会显示 “限时” 标签，营造紧迫感，促进用户快速领取。
关闭后则不显示该标签。
### 可叠加
设置优惠券的使用规则。
开启后，这张优惠券可以和其他优惠券同时使用，让用户享受更大力度的优惠。
关闭后，这张优惠券在同一订单中只能单独使用，避免平台过度让利。
