# 联系客服与反馈
作者：[阿城](https://blog.morehouse-s.com/)

## 启用“联系客服”模块
![](https://a1.boltp.com/2026/02/27/69a16c9601e60.png)
开启后，平台将在用户端（如个人中心、帮助中心、错误页面等）显示 “联系客服” 入口，用户可通过在线聊天、工单、QQ / 微信等方式与客服团队沟通；关闭后，所有客服入口将隐藏，用户无法直接联系客服。

## 显示左侧悬浮按钮
![](https://a1.boltp.com/2026/02/27/69a16d034a4e7.png)
开启后，在桌面端页面左侧会显示一条固定的悬浮条，包含 “联系客服” 和 “意见反馈” 两个核心入口；关闭后，该悬浮条将隐藏。

## 弹窗内容
![](https://a1.boltp.com/2026/02/27/69a16d7a1a7a8.png)
### 主标题
作为弹窗的核心吸引点。明确告知用户这里可以解决问题。

### 副标题
承接主标题，清晰地告诉用户下一步操作和能获得的服务。

### 二维码上方标题
在二维码最显眼的位置再次强化操作指令。

### 客服微信二维码
用户扫码后可以直接添加客服微信，实现一对一沟通。

### 二维码下方提示语
这是整个弹窗中最关键的 “信任背书”，直接打消用户对添加陌生微信的隐私顾虑。

### 弹窗底部按钮文案
引导用户在完成添加客服的操作后，主动关闭弹窗。

## 意见反馈入口

### 反馈接收邮箱
![](https://a1.boltp.com/2026/02/27/69a171b948ab0.png)
意见反馈邮箱的配置项，作用是接收用户通过 “意见反馈” 弹窗提交的内容。

## 反馈类型设置

### 反馈类型
![](https://a1.boltp.com/2026/02/27/69a172f9a90c5.png)
用于自定义用户在前端 “意见反馈” 窗口中看到的反馈分类选项。

### 图片最大尺寸
![](https://a1.boltp.com/2026/02/27/69a174f4d6ae3.png)
限制用户在平台上传单张图片的最大文件大小，单位为 MB（兆字节）。
