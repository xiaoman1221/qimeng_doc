# 邮件&SMTP
作者：[阿城](https://blog.morehouse-s.com/)

## 启用自定义 SMTP 发信
![](https://a1.boltp.com/2026/02/27/69a1bcbb1e8fb.png)
开启后，主题将使用下面配置通过 SMTP 发送邮件（包括意见反馈、系统通知等）。

## SMTP 服务器参数
![](https://a1.boltp.com/2026/02/27/69a1bd430c74e.png)
### SMTP 服务器地址
填写你所使用的邮箱服务商提供的 SMTP 发信服务器地址。

### SMTP 端口
SMTP 服务使用的网络端口，与加密方式对应。

### 加密方式
选择与端口对应的加密协议，确保邮件传输安全。

### SMTP 登录账号
登录 SMTP 服务器的账号，通常是你的完整邮箱地址。

### SMTP 密码 / 授权码
登录 SMTP 服务器的凭证。

### 发件人信息
![](https://a1.boltp.com/2026/02/27/69a1be293365e.png)

### 发件邮箱（From Email）
设置邮件中显示的 “发件人邮箱地址”，用户收到邮件时看到的发件邮箱就是这个。

### 发件人名称（From Name）
功能：设置邮件中显示的 “发件人名称”，用户收到邮件时看到的发件人名称就是这个。

## 测试工具

### 测试收件邮箱
![](https://a1.boltp.com/2026/02/27/69a1bf2dbe10d.png)
填写一个你能正常接收邮件的邮箱地址，作为测试邮件的接收方。

### 发送测试邮件
点击此按钮后，系统会使用你配置好的 SMTP 参数，向你填写的测试邮箱发送一封测试邮件。
