## 图片与媒体优化
![](https://a1.boltp.com/2026/02/27/69a1b7f092115.png)

### 启用图片懒加载
图片在即将滚动到视口才加载，而非页面一打开就全部加载，减少首屏网络请求数量。

### 懒加载应用范围：
大幅减少首屏加载的图片数量，降低带宽消耗，提升页面加载速度和用户体验。

### 启用 WebP 转换
自动将 JPG/PNG 等传统格式图片转换为更高效的 WebP 格式，在支持 WebP 的浏览器中使用，文件体积可减少 25%-50%。

### WebP 质量：
清除已生成的 WebP 缓存文件，当你替换了原图片或调整了质量设置后，需要点击此按钮重新生成 WebP 图片。


![](https://a1.boltp.com/2026/02/27/69a1b8d4201c2.png)
### 启用 DNS 预解析和预连接
提前解析页面中用到的第三方域名（如 CDN、统计、社交分享等）并建立网络连接，等真正需要加载资源时，就无需再等待 DNS 查询和连接建立的时间。

### 延迟加载非关键 CSS
对弹窗、模态框等非首屏显示的 CSS，使用 preload 技术延迟加载，避免这些样式阻塞首屏渲染，可节省约 100-200ms 的加载时间。

### 对主题内非关键 JS 使用 defer
让主题中大部分非核心的 JavaScript 脚本，在 HTML 解析完成后再执行，避免阻塞首屏渲染。

### 禁用 WordPress Emoji 脚本
移除 WordPress 自带的 Emoji 检测脚本和样式，这些脚本主要用于在老旧浏览器中渲染表情符号，现代浏览器已原生支持，移除后不影响正常表情使用。

### 禁用 oEmbed /iframe 自动嵌入
关闭 WordPress 自动将 YouTube、微博等链接转换为嵌入视频或卡片的功能，避免加载额外的 iframe 和相关脚本，减少若干 HTTP 请求。

### 列表 / 归档 / 搜索页每页文章数
统一控制首页、分类、标签、搜索等页面的每页文章显示数量，优先级高于 WordPress 全局设置。

### 限制 REST API 仅登录用户可访问
关闭对未登录用户的 REST API 公开访问，可有效降低恶意爬虫、探测流量对服务器的消耗。

## 缓存与 CDN 配置

### 启用静态资源 CDN 域名
https://a1.boltp.com/2026/02/27/69a1bb34da671.png
开启后，网站在前台输出图片、CSS、JS 等静态资源的 URL 时，会自动替换为你配置的 CDN 域名，让用户从离自己最近的 CDN 节点加载资源，而非直接从源服务器获取。

## 前端交互小优化

### 启用滚动/resize 事件节流
![](https://a1.boltp.com/2026/02/27/69a1bbdd025c2.png)
通过 requestAnimationFrame 等技术，对页面的滚动（scroll）和窗口大小调整（resize）这两个高频触发的事件进行 “节流”，避免事件回调函数被无限制地频繁执行。

### 启用 Turbo 性能优化
![](https://a1.boltp.com/2026/02/27/69a1bc46e3f53.png)
2用后，页面切换将使用 Turbo 加速，懒加载和脚本会在页面切换后自动重新初始化，动态内容可通过 AJAX 实时更新。
