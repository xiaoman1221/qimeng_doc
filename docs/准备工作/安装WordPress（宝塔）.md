# 安装WordPress
作者：[阿城](https://www.hidesg.ink/)

### 上传源文件
在网站页面里，点击网站根目录。
![wp_EnterSiteFolder.png](../../static/img/docs/4/wp_EnterSiteFolder.png)
删除文件
![wp_DeleteDefaultFiles.png](../../static/img/docs/4/wp_DeleteDefaultFiles.png)
上传WordPress源代码
![wp_UploadWordPressSourceCode.png](../../static/img/docs/4/wp_UploadWordPressSourceCode.png)
解压文件
![wp_UnzipWordPressFile.png](../../static/img/docs/4/wp_UnzipWordPressFile.png)
全部默认即可，直接解压
![wp_ConfirmUnzipWordPressFile.png](../../static/img/docs/4/wp_ConfirmUnzipWordPressFile.png)
解压完成后双击进入文件夹
![wp_EnterWordPressFolder.png](../../static/img/docs/4/wp_EnterWordPressFolder.png)
全选文件剪切。返回根目录。
![wp_CutAllFile.png](../../static/img/docs/4/wp_CutAllFile.png)
将文件粘贴在根目录下即可。

## 访问网站域名，进行下一步操作。

### 安装WordPress程序
当前页面下，点击“现在就开始！”按钮
![wp_StartInstall.png](../../static/img/docs/4/wp_StartInstall.png)
输入数据库名称密码进行安装必备库
![wp_InstallInputDataBaseInfo.png](../../static/img/docs/4/wp_InstallInputDataBaseInfo.png)
输入属于你自己的站点信息和管理员账号
![wp_InstallYourAdminNameAndPassword.png](../../static/img/docs/4/wp_InstallYourAdminNameAndPassword.png)
干得不错！来登录后台吧。
![wp_LoginToWordPress.png](../../static/img/docs/4/wp_LoginToWordPress.png)
