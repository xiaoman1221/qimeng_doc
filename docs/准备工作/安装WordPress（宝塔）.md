# 安装WordPress
作者：[阿城](https://blog.morehouse-s.com/)

### 上传源文件
在网站页面里，点击网站根目录。
![](https://a1.boltp.com/2026/02/24/699d673141f90.png)
删除文件
![](https://a1.boltp.com/2026/02/24/699d67314643f.png)
上传WordPress源代码
![](https://a1.boltp.com/2026/02/24/699d67313fc44.png)
解压文件
![](https://a1.boltp.com/2026/02/24/699d673144c2b.png)
全部默认即可，直接解压
![](https://a1.boltp.com/2026/02/24/699d6731322e2.png)
解压完成后双击进入文件夹
![](https://a1.boltp.com/2026/02/24/699d67314aa11.png)
全选文件剪切。返回根目录。
![](https://a1.boltp.com/2026/02/24/699d67345fc74.png)
将文件粘贴在根目录下即可。

## 访问网站域名，进行下一步操作。

### 安装WordPress程序
当前页面下，点击“现在就开始！”按钮
![1](https://a1.boltp.com/2026/02/23/699c2ed011bb5.png)
输入数据库名称密码进行安装必备库
![1](https://a1.boltp.com/2026/02/23/699c2ecfd1723.png)
输入属于你自己的站点信息和管理员账号
![1](https://a1.boltp.com/2026/02/23/699c2ed0224e6.png)
干得不错！来登录后台吧。
![1](https://a1.boltp.com/2026/02/23/699c2ecfc7eef.png)
