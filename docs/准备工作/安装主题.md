# 安装主题
推荐阅读时间：5分钟 作者：[阿城](https://blog.morehouse-s.com/)


进入仪表盘，点击左侧菜单的外观选项。
![1](https://a1.boltp.com/2026/02/23/699c4bed58f39.png)
点击主题页面的安装主题
![1](https://a1.boltp.com/2026/02/23/699c4bed5fa93.png)
在刷新的页面里点击 上传主题
![1](https://a1.boltp.com/2026/02/23/699c4bed7bcec.png)
然后的 上传主题 中，选中 qimeng 主题安装
![1](https://a1.boltp.com/2026/02/23/699c4bed58f0c.png)
安装之后，启用主题
![1](https://a1.boltp.com/2026/02/23/699c4bed3b707.png)
