# 安装主题
作者：[阿城](https://www.hidesg.ink/)

进入仪表盘，点击左侧菜单的外观选项。
![theme_StartInstall.png](/img/docs/4/theme_StartInstall.png)
点击主题页面的安装主题
![theme_ClickInstallThemeButton.png](/img/docs/4/theme_ClickInstallThemeButton.png)
在刷新的页面里点击 上传主题
![theme_UploadTheme.png](/img/docs/4/theme_UploadTheme.png)
然后的 上传主题 中，选中 qimeng 主题安装
![theme_SelectThemeFile.png](/img/docs/4/theme_SelectThemeFile.png)
安装之后，启用主题
![theme_EnableTheme.png](/img/docs/4/theme_EnableTheme.png)
