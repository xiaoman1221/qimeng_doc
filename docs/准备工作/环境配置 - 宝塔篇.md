# 环境配置 - 宝塔篇
作者：[阿城](https://blog.morehouse-s.com/)

### PHP配置
点击进入软件商店
![](https://a1.boltp.com/2026/02/24/699d4a58d026c.png)
在搜索框中搜索 PHP。
![](https://a1.boltp.com/2026/02/24/699d4a58eba9b.png)
PHP版本选择 PHP 8.1版本，选择完成点击极速安装。
![](https://a1.boltp.com/2026/02/24/699d4a58dfdc8.png)
进入 已安装 分类，选择并进入PHP 8.1 设置
![](https://a1.boltp.com/2026/02/24/699d4a58de56d.png)
在PHP设置中点击安装扩展，选择并安装ionCube 扩展，耐心等待绿色字样的安装变成红色字样的卸载，即为安装成功。
![](https://a1.boltp.com/2026/02/24/699d4a58dec3a.png)
点击服务，点击重启。以重启PHP服务。
![](https://a1.boltp.com/2026/02/24/699d4a58e6e26.png)
