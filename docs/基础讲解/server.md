# 服务器的选择和购买

作者：[小满1221](https://www.yhdzz.cn/)


第一次购买服务器肯定会出现迷茫和不知道咋选的情况，对于新手来说，大家通常是没有ICP备案的，所以建议购买中国大陆以外的服务器，例如：中国香港，美国或者欧洲的服务器。

这部分需要一些耐心，还需要有一些英文基础，或者配备浏览器翻译插件等，因为大部分国外厂商的语言都是英文

第一次的选择不要给自己太大压力，因为往往你的第一个选择都是错误的，所以不需要说一定要选对选好，放心大胆的选择，当然，目前国内外都有很多免费试用的服务器或者虚拟主机，所以试错成本很低。

大厂，诸如：阿里云，腾讯云，华为云或者国外的亚马逊，谷歌云等平台，会给你一个不错的体验，这些大厂通常配置简单易用，而且都有中文的支持。

小厂，诸如：小兔云，这些小厂通常配置简单易用，很便宜，而且都有中文的支持。

**声明：笔者推荐的厂商均无广告，选择什么看你，而且小厂的稳定性和大厂的稳定性不同，必须仔细斟酌！！！**
