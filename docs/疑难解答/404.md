# 页面显示404 Not Found
作者：[小满1221](https://www.yhdzz.cn/)

## 原因

1. 没设置伪静态，参考先前教程设置
2. 没有文件
3. 主题是不是没激活
4. 网站文件夹权限
5. WordPress内固定连接设置无效或未设置

## 解决方案

1. 设置伪静态
2. 在后台查看是否文件没在网站，或者是解压的时候解压到别的目录了
3. 安装主题后如果出现此情况说明主题没授权激活，进入后台填写授权码完成授权操作即可。
4. 设置文件目录权限支持Nginx和PHP读取/执行（一般用户为www和www-data 一般权限为755）
5. 在WordPress内固定连接设置中设置你的固定连接，推荐：`/%post_id%.html`
